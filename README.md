## Installation
Code for backed part of data visualization project.

## Tech Stack
- NestJS
- Prisma
- PostgreSQL
- TypeScript
- Vercel Blob Storage for object storage
- Vercel Postgres for database

## Feautures
- [x] Pagination, filtering, sorting for dataset
- [x] CRUD for dataset
- [x] Upload dataset to object storage

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
